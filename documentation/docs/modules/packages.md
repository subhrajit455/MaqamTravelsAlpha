# Packages Module

The `packages` module manages bundle deals, combining flights, hotels, tours, and transport options into unified package itineraries.

## Component Map

* **Routes:** `package.routes.js`
* **Controller:** `package.controller.js`
* **Service:** `package.service.js`
* **Model:** `Package`

---

## Core Operations

### 1. Package Catalog
* Supports listing pre-defined itineraries (e.g. Umrah packages, cultural packages).
* Contains flight schedules, hotel selections, transfer services, visa processing details, and guidance services.

### 2. Custom Packages (CRM Interface)
* Sales agents can generate custom package variations for corporate clients or large families using the operational CRM endpoints, including custom markups and discounts.
